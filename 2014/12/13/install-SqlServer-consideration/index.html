帮同学安装SqlServer感悟

发表于 2014-12-13   |   分类于 数据库   |  

背景

  • 安装vs2013之后卸载而且没卸载干净。
  • 安装vs2012，可以使用。
  • 安装sqlserver2008R2没成功，系统残留很多文件和没有清理的注册表。
  • 安装sqlserver2012失败，残留注册表等文件。

我的做法

  • 查找方法彻底删除sqlserver残留文件。点击查看链接
  • 安装sqlserver2012安装教程

遇到问题


  • 第一次安装我没有在意这个问题，直接确定忽略继续安装，安装好之后问题来了:没有Microsoft Sqlserver Manager Studio这个软件，后来才知道因为没有权限读取注册表所以这个软件无法安装。这是不要担心，运行微软的安装程序点击“安装全新的数据库或者添加功能”，继续安装，到时候点击添加功能把没安装的软件安装上就好了
  • 你以为这样就结束了吗！下面才是噩梦的开始！每次添加软件都会提示：没有权限读取…注册表。解决权限无法读取注册表用这个方法把无法读取的注册表添加权限就可以了。
  • 再次安装又报同样的错误！只是不能读取的注册表换了，没事儿，按照上面的方法把权限加上去就行了。就这样来回4、5次我就放弃了，因为安装的文件确实太多，我是在是受不了了。不知道还有多少注册表，所以果断重装系统！

总结

  • 遇到这样的问题千万不要犹豫，直接重装系统，先破后立，从头再来。以前的问题就不要管了，因为根本就管不了！！！
