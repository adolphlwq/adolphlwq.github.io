pre-knowlege

HTML

  • HTML 是用来描述网页的一种语言
  • HTML 指的是超文本标记语言 (Hyper Text Markup Language)
  • HTML 不是一种编程语言，而是一种标记语言 (markup language)
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页

CSS

  • Cascading[kæ’skeɪdɪŋ] Style Sheets：层叠样式表、级联样式表
  • 样式定义如何显示 HTML 元素、优化网页的表现形式
  • 样式通常存储在样式表中
  • 把样式添加到 HTML 4.0 中，是为了解决内容与表现分离的问题
  • 外部样式表通常存储在 CSS 文件中

XML

  • XML 指可扩展标记语言（EXtensible Markup Language）
  • XML 是一种标记语言，很类似 HTML
  • XML 的设计宗旨是传输数据，而非显示数据
  • XML 标签没有被预定义。您需要自行定义标签。
  • XML 被设计为具有自我描述性。
  • XML 是 W3C 的推荐标准

css渲染xml和html是通过类型选择器实现的

id选择器

类选择器 VS ID选择器(HTML 和 XHTML 将类和 ID 值定义为区分大小写)

  • 区别 1：只能在文档中使用一次与类不同，在一个 HTML 文档中，ID 选择器会使用一次，而且仅一次。
  • 区别 2：不能使用 ID 词列表:不同于类选择器，ID 选择器不能结合使用，因为 ID 属性不允许有以空格分隔的词列表。
  • 区别 3：ID 能包含更多含义

属性选择器

标签是用来标记内容的，在html的标签中，不同的标签都有自己的属性，比如<a>标签可以有href,title,target等属性，属性选择器就是通过标签的属性来进一步选择。
demo:

只有具备href属性的a标才会变成红色

同时具备href和name属性的a标签才会变成红色

只有具备href和 title且href=”http://www.w3school.com.cn/",title="W3School"的标签才会变成红色。
需要注意的是：属性和属性值必须完全匹配
p.important 和 p[class=”important”]等价

部分属性选择器

p[class~="important"] {color: red;}
只要属性中包含important，就选择。
“~=” 属性选择器能用于任何属性，而不只是 class(ID)

子串匹配属性选择器

更高级的选择器模块,它是 CSS2 完成之后发布的。简单总结：

type description
abc^=”def”] 选择 abc 属性值以 “def” 开头的所有元素
abc$=”def”] 选择 abc 属性值以 “def” 结尾的所有元素
abc*=”def”] 选择 abc 属性值中包含子串 “def” 的所有元素

举例来说，如果希望对指向 W3School 的所有链接应用样式，不必为所有这些链接指定 class，再根据这个类编写样式，而只需编写以下规则：

a[href*="w3school.com.cn"] {color: red;}

后代选择器

h1 em{color:red;}
选择h1后包含的所有em属性，中间必须有一个空格

子元素选择器

h1 > strong {color:red;}

相邻兄弟选择器

要求：

  • 要选择的元素紧接在另一个元素后的元素
  • 而且二者有相同的父元素

h1 + p {margin-top:50px;}
请记住，用一个结合符只能选择两个相邻兄弟中的第二个元素

伪类（pseudo-classes)

属性 描述 CSS
:link 向未被访问的链接添加样式。 1
:visited 向已被访问的链接添加样式 1
:hover 当鼠标悬浮在元素上方时，向元素添加样式 1
:active 向被激活的元素添加样式 1
:foucs 向拥有键盘输入焦点的元素添加样式 2
:first-child 向元素的第一个子元素添加样式 2
:lang 向带有指定 lang 属性的元素添加样式 2

伪元素（pseudo-elements)

属性 描述 CSS
:first-line 向文本的第一个字母添加特殊样式 1
:first-letter 向文本的首行添加特殊样式 1
:before 在元素之前添加内容 2
:after 在元素之后添加内容 2
